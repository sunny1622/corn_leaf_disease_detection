# Corn Leaf Disease Detection 🌽

## Project Description

Corn Leaf Disease Detection is a web-based application that uses machine learning to identify diseases in corn leaves from images. The system helps farmers detect plant diseases early and take preventive actions to improve crop health and yield.

## Features

* Upload corn leaf images
* Detect disease using a trained machine learning model
* User authentication (login and registration)
* Admin and user dashboards
* Display prediction results

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Machine Learning
* TensorFlow / Keras

## How to Run the Project

1. Clone the repository
2. Install required libraries:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open the browser and go to:

```
http://127.0.0.1:5000
```

## Purpose

The main goal of this project is to help farmers and agricultural experts identify corn leaf diseases quickly using artificial intelligence.
